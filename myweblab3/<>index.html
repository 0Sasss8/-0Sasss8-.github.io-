什麼是三角函數？

三角函數是數學中用來描述角度與邊長關係的函數，特別是在直角三角形中。常見的三角函數包括正弦 (sin)、餘弦 (cos) 和正切 (tan)。這些函數用來表示一個角的對應邊長比例。

正弦 (sin)

在一個直角三角形中，正弦函數用來描述對應角的對邊與斜邊的比值。

公式： sin(θ) = 對邊 / 斜邊

例如：在一個有30度角的三角形中，sin(30°) = 0.5。

餘弦 (cos)

餘弦函數用來描述對應角的鄰邊與斜邊的比值。

公式： cos(θ) = 鄰邊 / 斜邊

例如：在一個有60度角的三角形中，cos(60°) = 0.5。

正切 (tan)

正切函數用來描述對應角的對邊與鄰邊的比值。

公式： tan(θ) = 對邊 / 鄰邊

例如：在一個有45度角的三角形中，tan(45°) = 1。
